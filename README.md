# AwA - another weather app

## О чём приложение? 
В широком доступе уже доступно множество приложений, которые так или иначе показывают погоду. Прогнозы всегда красивые, подробные и даже содержат некоторую подозрительно специфичную информацию. Но это всё довольно скучно. Это приложение предлагает (или собирается предложить) нестандартный погодный функционал
- Погода не там где ты, а там где твои друзья ***UPD: уже реализовано***
- Уведомления о резкой смене погоды
- Живые обои, меняющиеся в зависимости от погоды (под вопросом, но звучит интересно) ***UPD: этой фичи не будет, автор передумал***

## Кому это надо? 

Вот несколько соверщенно обычных ситуаций, в которых нужны необычые фичи

```
As a (Как): Геймер (или друг, который хочет поболтать в Discord)
I want to (Я хочу): Видеть список друзей с иконками их текущей погоды на одном экране
So that (Чтобы): Быстро понять, кто из них сейчас скорее всего сидит дома (из-за плохой погоды) и готов пойти в онлайн-игру, не опрашивая каждого лично.
```

```
As a (Как): Заботливый друг
I want to (Я хочу): Получать пуш-уведомление, когда в городе моего друга начинается ливень или сильный снег
So that (Чтобы): У меня был повод написать ему сообщение («Не забудь зонт!» или «Ну как ты там?») и проявить внимание, не выглядя навязчивым.
```

```
As a (Как): Активный пользователь, часто меняющий локации
I want to (Я хочу): Чтобы приложение автоматически обновляло мою погоду и город для моих подписчиков в фоновом режиме
So that (Чтобы): Мои близкие знали, в каких условиях я нахожусь, и не волновались (или не предлагали погулять в парке, когда у меня ураган), без необходимости мне писать об этом вручную.
```

- Firebase
