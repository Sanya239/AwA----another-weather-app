# Гипотезы:
1. Информация о погоде соответствует запросам пользователей
2. Система друзей эффективна
3. Настройки и редактирование профиля удобны
4. Все функции приложения интуитивно понятны и доступны

# Исследования:
### Проблемное интервью
**Гипотеза:**
- Продукт представляет интерес для пользователей

**Вопросы:**
- Когда в последний раз погода у друга была для тебя важна?
- Бывало ли, что ты писал: “как у вас там?
- Были ли случаи, когда вы хотели поделиться своей погодой, но это было слишком долго делать через скриншот?
- Насколько комфортно вам делиться своим местоположением (до района/улицы) с друзьями в реальном времени?

**Выводы:**
- Проблема, решаемая рассматриваемым продуктом, актуальна


### Интервью по Jobs to be Done
**Сравниваем AwA против альтернатив:**
- Нагуглить погоду 
    - AwA проигрывает по подробности данных
    - AwA выигрывает по скорости и удобству
- Спросить друга
    - Минусов нет, этот спосбо нужно использовать в связке с AwA

### Метод Кано
**Предлагаемые фичи**
- Уведомления  – must have (реализовано)
- Реакции   – wow-effect (не реалиовано)
- deep-links – wow-effect (не реализовано)
- геймификация – reverse (не будет реализовано)

### 5-секундный тест
**Гипотеза:**
- Cоциальный контекст приложения сразу распознаётся пользователем

**Проверка:**
- Основной экран приложения показывается пользователю на 5 секунд.

**Результат:**
- Все (оба) протестированных участника за 5 секунд уловили суть и задачи продукта

Исправления по итогам всех исследований:
1) Добавлено поле “ощущается как”
1) Сортировка друзей по температуре
2) Кнопки удалить друга и отменить поменяли приоритеты
3) Клавиатура скрывается после успешного сохранения профиля
3) Подтверждение выхода из аккаунта
4) Добавлена подсказка в диалог добавления друга
4) Кнопка назад пускает назад
4) Иконка приложения более контрастная
